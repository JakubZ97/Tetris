## Tetris
A clone of tetris, written in vanilla JS, CSS and HTML.  
Only playable on PC.  
  
Key binding:  
<kbd>←</kbd>/<kbd>→</kbd> - moves block horizontally  
<kbd>↑</kbd> - rotate    
<kbd>↓</kbd> - soft drop  
<kbd>space bar</kbd> - hard drop  
  
You can try it [here](https://jakubz97.github.io/Tetris/).
